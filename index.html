Kommentare

Damit $\LaTeX$ einen Befehl von Text unterscheiden kann müssen diese mit einem \ Zeichen angekündigt werden. Um z.B. den Befehl command zu benutzen schreibt man im $\LaTeX$ Dokument \command.
Damit der Befehlt allerdings weiß, auf welchen Text er sich beziehen soll, wird der Text (auch Parameter genannt) mit geschweiften Klammern ({}) umschlossen. Daraus ergibt sich die folgende Syntax \command{parameter}.
Mit diesem Wissen können die meisten Befehle bereits benutzt werden, manche Befehle können allerdings auch Optionen annehmen. Diese werden mit eckigen Klammern [] angekündigt: \command[option=value]{parameter}.

Beispiel

  • Aus \textit{Kursiver Text} wird: $\textit{Kursiver Text}$
  • Aus \textbf{Fetter Text} wird: $\textbf{Fetter Text}$
  • Aus \underline{Unterstrichener Text} wird: $\underline{Unterstrichener Text}$

Grundlegende Dokumenten-Struktur

Damit der Compiler ein $\LaTeX$ Dokument zu ein PDF konvertieren kann muss ein gewisses Grundgerüst vorhanden sein: 


\documentclass{article}
\usepackage[utf8]{inputenc}

\title{Beispiel}
\author{Max Mustermann}
\date{August 2021}

\begin{document}

\maketitle

\end{document}

Sie müssen den Source Code mit \documentclass{} starten. Darauf folgen Pakete, die sie Einbinden und das setzen von Variablen, welche sie benötigen (z.B. Maketitel).
Darauf folgt der \begin{document} ... \end{document} Block. Hier schreiben sie den Text hinein, den sie anzeigen lassen wollen/ was sie anschließen im PDF sehen können.

Dokumentenklasse

\documentclass[option1=value1, option2=value2]{class_name}

In dem Beispiel aus Kapitel 2 wird in der ersten Zeile eine Dokumentenklasse definiert. Zur Auswahl stehen article, proc, minimal, report, book, slides, memoir, letter und beamer. Sie definieren das grundlegende Aussehen des Dokuments, für eine Liste von Klassen und eine Beschreibung der Funktion klicken sie bitte hier.

Pakete einbinden

\usepackage[option1=value1, option2=value2]{package_name}

Es ist möglich in $\LaTeX$ Pakete einzubinden, um gewisse Extrafunktionalitäten zu erhalten.
Ein beliebtes Beispiel ist die Funktionalität um Grafiken einzubinden. Hierfür müssen Sie per Befehl $\LaTeX$ informieren, dass es das graphicx Paket einbinden soll. Der Befehlt wäre \usepackage{graphicx}.
Manche Pakete benötigen Optionen, damit sie wissen wie die Extrafunktionalität bereitgestellt werden soll. Ein solches Paket haben sie bereits kennengelernt: inputenc, \usepackage[utf8]{inputenc}. Das Paket sorgt dafür, dass (im Deutschen) Umlaute erkannt und verwendet werden können, bzw. alle Zeichen aus dem UTF-8 Datensatz (dies umfasst auch Indische, Chinesische, … Schriftzeichen).

Beachten sie, dass Pakete nur vor \begin{document} eingebunden werden können.

Welches Packet wie eingebunden werden muss, und welche Optionen zur Verfügung stehen müssen Sie allerdings selbstständig recherchieren.

Maketitle



\title{Beispiel}
\author{Max Mustermann}
\date{August 2021}

...

\maketitle

Um ein Titelblatt zu definieren brauchen sie kein Paket (sie benötigen Pakete, wenn sie das Aussehen des Titelblattes anpassen wollen). $\LaTeX$ hat die vorgefertigt Funktion \maketitle, welche keine Parameter nimmt. Stattdessen müssen sie die Parameter selbst mit \title{}, \author{} und \date{} setzen.

Umbrüche


\newline
\\
\newpage
\par

Das einfügen von Seitenumbrüchen, bzw. Zeilenumbrüchen ist sehr einfach.

Sie können \newline für einen Zeilenumbruch verwenden, da Zeilenumbrüche allerdings sehr häufig vorkommen wurde eine Abkürzung eingefügt: \\. Beide Ausdrücke sind equivalent.

Benutzen sie \newpage um einen Seitenumbruch zu erstellen.

Wenn sie einen Absatz erzeugen wollen, dann können sie \par benutzen, oder aber sie lassen in ihrme Source Code eine Zeile frei (erstellen in ihrem Source Code eine Absatz).

Mathematik in $\LaTeX$

Damit sie Formeln, oder Rechnungen in $\LaTeX$ schreiben können müssen sie diese lediglich mit $ Zeichen ummanteln. Wenn sie etwas in dieser Schreibweise schreiben, dass wird das Mathe-Modus genannt.
Wollen sie z.B. a+b=c darstellen, dann schreiben sie $a+b=c$ (das ergibt $a+b=c$).

Die einfache Mathematik kann mit den Symbolen dargestellt werden, die sie auf der Tastatur schreiben können. Wollen Sie allerdings erweiterte Symbole benutzen gibt es von $\LaTeX$ vordefinierte Befehle. Bitte beachten sie, dass diese Befehle nur im Mathe-Modus verwendet werden können.
